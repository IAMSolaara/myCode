per poter usare questa roba serve avere Qt5. Non funziona su Qt4
oltre a questo serve PyQt5 (che ha la versione LGPL di Qt5 integrata), che si puo' installare con `pip3 install -r reqs.txt` in questa cartella, il che sottintende che ci sia anche python3
per avviare questo coso basta fare `python3 main.py`
